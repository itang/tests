# test moquette mqtt

## usage

run server

```
$ gradle run
```

run client1(publish)

```
$ gradle run_client1
```

run client2(subscibe)

```
$ gradle run_client2
```

run client3(listener disconnnect event)

```
$ gradle run_client2
```

run client4(async)

```
$ gradle run_client4
```
